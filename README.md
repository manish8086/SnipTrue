# SnipTrue 🛡️

**SnipTrue** is an AI-powered Chrome Extension designed to combat misinformation and media bias in real-time. By leveraging advanced Natural Language Processing (NLP), SnipTrue analyzes news articles and social media posts as you browse, providing instant insights into factual accuracy and potential political leaning.

[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](http://makeapullrequest.com)

## ✨ Features

- **Real-time Bias Detection:** Analyzes the sentiment and tone of articles to identify political or ideological leanings.
- **AI Fact-Checking:** Cross-references key claims against a database of verified information using state-of-the-art LLMs.
- **Source Reliability Scores:** Provides a "Trust Rating" for the publisher based on historical reporting accuracy.
- **Contextual Summaries:** Generates concise, objective summaries of long-form content to help users grasp the core facts quickly.
- **Non-Intrusive UI:** Operates as a discreet sidebar or overlay that only appears when you need it.
